AI Search 🚀

A modern, AI-powered search interface featuring cutting-edge Glassmorphism design trends for 2025. This application provides instant AI-generated answers to any question using Google's Gemini AI model, wrapped in a stunning frosted-glass user interface.

✨ Features

- AI-Powered Search: Get intelligent answers to any question using Google's Gemini 1.5 Flash model
- 2025 Glassmorphism Design: Modern frosted-glass UI with backdrop blur effects and transparency layers
- Responsive Design: Optimized for both desktop and mobile devices
- Real-time Interaction: Instant search results with smooth animations
- Accessibility Features: High contrast modes and WCAG 2.1 compliant design elements
- Premium Visual Effects: Multi-layered glass panels, floating elements, and subtle animations

🎨 Design Highlights

This application showcases the latest Glassmorphism trends dominating 2025 UI design:

- Transparent Panels: Semi-transparent elements with backdrop-filter: blur() effects
- Layered Depth: Multi-dimensional glass surfaces with realistic shadows and borders
- Interactive Elements: Smooth hover effects and responsive feedback
- Modern Color Palette: Gradient backgrounds with sophisticated glass refraction
- Floating Glass Orbs: Ambient decorative elements that enhance the immersive experience

🚀 Technologies Used

- HTML5: Semantic markup with modern viewport configuration
- CSS3: Advanced glassmorphism effects using backdrop-filter and CSS gradients
- JavaScript ES6: Modular architecture with ES6 imports
- Google Generative AI: Gemini 1.5 Flash model for intelligent responses
- Responsive Design: Mobile-first approach with media queries

📋 Prerequisites

- Modern web browser with ES6 module support
- Google Generative AI API key
- Internet connection for API calls

⚙️ Installation & Setup

1. Clone the repository:
   git clone https://github.com/yourusername/ai-search-glassmorphism.git
   cd ai-search-glassmorphism

2. Get your Google Generative AI API key:
   - Visit Google AI Studio at https://makersuite.google.com/app/apikey
   - Create a new API key
   - Copy your API key

3. Configure API Key:
   - Open the HTML file
   - Replace "YOUR_API_KEY_HERE" with your actual Google Generative AI API key:
   let api_key = "YOUR_ACTUAL_API_KEY_HERE";

4. Launch the application:
   - Open index.html in your web browser
   - Or serve it using a local server for optimal performance

💡 Usage

1. Ask Questions: Type any question in the search input field
2. Get AI Answers: Click "Click to Search" or press Enter to get instant AI-generated responses
3. Explore Interface: Experience the glassmorphism effects through hover interactions and animations

Example Queries:
- "What is artificial intelligence?"

🔧 Customization

Modify Glassmorphism Effects:
/* Adjust transparency levels */
background: rgba(255, 255, 255, 0.08); /* Change opacity */

/* Modify blur intensity */
backdrop-filter: blur(20px); /* Adjust blur amount */

/* Update glass borders */
border: 1px solid rgba(255, 255, 255, 0.2); /* Border opacity */

Change AI Model:
Replace "gemini-1.5-flash" with other available Gemini models in the JavaScript section.
